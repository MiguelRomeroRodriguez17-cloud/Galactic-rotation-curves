# Curvas de rotación galácticas: Inferencia bayesiana de parámetros de materia oscura

## Descripción
Este proyecto aplica inferencia bayesiana y MCMC para estimar parámetros de materia oscura a partir de curvas de rotación galácticas, utilizando el modelo de disco estelar exponencial más un halo NFW.

## Contenido del repositorio
- Cuaderno de Colab con todo el código
- Archivo NGC3198_rotmod con los datos de la galaxia analizada
- Carpeta con las figuras generadas en el cuardeno de Colab
- Carpeta con los resultados generados en el cuardeno de Colab
- Carpeta con el archivo main.text LaTeX y sus figuras correspondientes

## Requisitos para ejecutar el código
- Python 3.8+
- Importar bibliotecas (numpy, scipy, matplotlib, emcee, corner, pandas)

## Resultados principales
- Simulación valida de los métodos utilizados (recupera parámetros verdaderos)
- NGC 3198: M_d = 2.83e10 M_sol, R_d = 4.14 kpc, M_200 = 5.31e11 M_sol, c = 8.17
